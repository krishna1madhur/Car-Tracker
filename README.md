# car-tracker

## technologies used:
** Java
** Spring MVC
** MySQL
** Angular JS

## directory structure:

**`client`** [*module-client*]: contains ui app (HTML, CSS, JS, fonts, images)      
**`api`** [*module-api*]: contains REST API
